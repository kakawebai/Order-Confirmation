Google Programmable Search

API Key 与 CX 获取指南

用于 Google Custom Search API（Programmable Search Engine）调用

你需要两样东西

API Key（身份密钥） + CX（搜索引擎 ID）

打开 Cloud Console 打开 PSE 控制台
第一步
获取 API Key
  1. 1
    访问 Google Cloud Console。
  2. 2
    创建项目：点击左上角项目选择器 → “新建项目” → 命名后保存。
  3. 3
    启用 API：在顶部搜索栏输入 Custom Search API → 进入后点击 “启用（Enable）”。
  4. 4
    生成凭据：侧边栏 “API 和服务” → “证书” → “+ 创建凭据” → 选择 “API 密钥”。
  5. 5
    系统会弹出一个字符串（例如 AIzaSy...），这就是你的 API Key，请妥善保存。
安全建议
  • 把 API Key 限制为仅允许 Custom Search API。
  • 尽量限制来源（HTTP referrers / IP）以降低泄露风险。
  • 不要把 Key 写进公开网页或公开仓库。
第二步
获取 CX（搜索引擎 ID）
  1. 1
    访问 Programmable Search Engine 控制面板。
  2. 2
    点击 “开始使用” 创建新的搜索引擎。
  3. 3
    关键设置：搜索引擎名称随意；搜索范围网址请填写 www.google.com（仅为通过初始化验证，后续会开启全网搜索）。
  4. 4
    开启全网搜索：创建后进入 “概览（Overview）” → 打开 “搜索网页（Search the entire web）” 开关。
  5. 5
    复制 ID：在同一页面找到 “搜索引擎 ID”（例如 a1b2c3d4e5f6g7h8i），这就是你的 CX。
快速自检
  • 如果没开启 “Search the entire web”，结果会只在你配置的站点内。
  • CX 和 API Key 都准备好后，才能稳定调用 Custom Search API。
